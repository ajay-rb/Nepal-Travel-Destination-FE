### 🚀 **Steps to Start the Project**

1️⃣ **Clone the Repository**  
```sh
git clone 
```

2️⃣ **Install Dependencies**  
```sh
npm install
```
or  
```sh
yarn install
```

3️⃣ **Run the Development Server**  
```sh
npm run dev
```
or  
```sh
yarn dev
```

4️⃣ **Open in Browser**  
Go to **`http://localhost:5173`**  

🔥 You're all set! Happy coding! 🎉
